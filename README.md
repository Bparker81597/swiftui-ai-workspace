# SwiftUI AI Workspace

A polished SwiftUI dashboard concept for reviewing AI-assisted coding sessions, tracking outputs, and submitting structured feedback reports.

## Overview
This project simulates a real-world workflow for evaluating AI-generated code. It focuses on clean UI design, reusable components, and structured feedback systems.

## Features
- Dashboard for active AI coding sessions
- Session detail view with evaluation actions
- Structured feedback reporting form
- Reusable SwiftUI component library
- Clean card-based UI with strong visual hierarchy
- Mock data to simulate real product usage

## Screens
- Dashboard
- Session Detail
- Feedback Report
- Component Library

## Tech Stack
- Swift
- SwiftUI
- Combine

## Focus Areas
- UI/UX polish and consistency
- Component reusability
- Realistic product workflows
- Clean architecture (Models / ViewModels / Views)

## Why I Built This
I built this project to strengthen my SwiftUI skills and explore UI patterns for AI-assisted development workflows, including evaluation, reporting, and session tracking.
